## Train Info Kiosk

This repository contains a lightweight train transport kiosk that serves a static passenger information web UI locally and displays it in Chromium fullscreen kiosk mode.

It is designed for station screens, onboard displays, and transport terminals, providing a reliable way to run a web-based interface in a locked-down fullscreen environment.

---

##  Features

- Fullscreen Chromium kiosk mode
- Local static web server (offline-friendly)
- Simple deployment for dedicated kiosk machines
- Suitable for train passenger info, schedules, and service updates
